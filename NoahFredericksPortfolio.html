Projects

Image Reconstruction

This machine learning program utilizes Lasso regression and cross-validation techniques to create a sparse yet accurate model of an image based on a small proportion of sampled pixels. This model can be employed to reconstruct an image with a large number of corrupted pixels.

Brain-Computer Interface Movement Decoding

Using a Support Vector Machine and cross-validation, this program classifies movement data acquired from a Brain-Computer Interface, effectively differentiating between mental decisions of "left" and "right". This model's functionality can extend to encoding bionic limbs or brain-controlled robots, enabling them to respond according to desired commands while utilizing a minimal number of electrodes attached to the user.

Economic Analysis and Visualization

This comprehensive program processes, analyzes, and visualizes a diverse range of crucial economic features obtained through the FRED API. It offers a streamlined interface that serves as a convenient alternative to navigating https://fred.stlouisfed.org for accessing and utilizing this data.

Financial Sector Evaluation

This program will utilize data from the Economic Analysis and Visualization program to create a regression model utilizing cross-validation. By leveraging the yfinance library to generate target indexes, it will highlight the influence of these features on different sectors of the economy. It should be able to be employed to identify significant macroeconomic factors impacting each industry and extrapolate their market value based on these features.
