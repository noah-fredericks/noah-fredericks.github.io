Economics Programming Project

Data Analysis and Visualization

Project Description

This comprehensive program processes, analyzes, and visualizes a diverse range of crucial economic features obtained through the FRED API. It offers a streamlined interface that serves as a convenient alternative to navigating https://fred.stlouisfed.org for accessing and utilizing this data.

This task required accessing the FRED API using the requests library and an appropriate API key. A dictionary of the series ID and the name of the series was used to group the names of each series, with its ID within the API. After creating a formatted url with the appropriate parameters, the formatted data was merged with a pandas dataset on the 'date' column, which was renamed according to the name of each series, and interpolated to fill in gaps in the data.
